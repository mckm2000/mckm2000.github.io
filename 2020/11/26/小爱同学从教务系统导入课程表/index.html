不务正业系列之“小爱同学从教务系统导入课程表”

关于这个的开发其实小爱同学本身已经提供了工具，工具自带的函数scheduleHtmlProvider已经可以完美抓取我们学校的教务系统页面了，主要的工作还是编写scheduleHtmlParser函数。而scheduleHtmlParser函数的作用就是返回课程信息，返回的值有固定的数据结构，所以在scheduleHtmlParser中我们主要是想办法从获取到的页面中提取出课程信息，然后重组数据，再按照他的数据结构重组数据。

这是课程表的表格，我一开始想的是用$(#kbtable)获取表格，然后一路find()到具体课程，然后再解析数据。但是我发现教务系统的代码有一些问题，他的老师之类的数据是有<font>包着的，但是课程名没有，就像下面这样。

一开始是靠获取整个div的text然后再删去div里所有font的文本，就之剩下课程名了。

但是后来我发现，事情并没有这么简单，因为课表中还有同一个时间不同周上不同课的，然后他竟然是用一个横线隔开就没了。

然后后面就想获取html源码然后用split()来分割，结果后面发现html()获取到的是未解码的代码。然后在一位大佬的指点下，通过增加了一行var $ = cheerio.load(html, { decodeEntities: false })解决了这个问题。
