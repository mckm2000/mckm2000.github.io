扩展欧几里得算法

扩展欧几里得定理：对于不完全为 0 的整数 a，b，gcd（a，b）表示 a，b 的最大 公约数。那么一定存在整数 x，y 使得 gcd（a，b）=ax+by（贝祖等式）。

扩展欧几里得算法可用于求解贝祖等式中的x，y。

根据欧几里得算法（gcd)的性质：gcd(a,b) = gcd(b,a%b) = gcd(a%b,b%a%b) = ...

贝祖等式可拆分为：$$ax_1$$+$$by_1$$ = $$bx_2$$+(a%b)$$y_2$$ = b$$x_2$$ + (a-(a/b)*b)$$y_2$$ = …​

合并同类项后得到：a($$x_1$$-$$y_2$$)+b($$y_1$$-[$$x_2$$-(a/b)*b$$y_2$$]) = 0

根据恒等定理 $$x_1$$ = $$y_2$$,$$y_1$$ = ($$x_2$$-(a/b)$$y_2$$)
